<h2>Requirements</h2>

* Python3
<h2>How to install</h2>
Download this repository to your computer  

```console
foo@bar:~$ git clone https://github.com/cl1ckname/CookJoke.git
```
Go to repository folder  

```console
foo@bar:~$ cd CookJoke/
```

Run `example.py`  
```console
foo@bar:~$ python3 src/example.py
```

